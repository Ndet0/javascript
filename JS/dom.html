The most common types are element nodes, text nodes, and attribute nodes. 2.Tree Structure: The DOM represents the document as a hierarchical tree structure where each node is connected to its parent and child nodes. 3.Methods and Properties: The DOM provides various methods and properties to access and manipulate nodes. Common properties for traversing: (a)parentNode (b)childNodes: 1.firstChild 2.lastChild (c)nextSibling (d)previousSibling Modifying Elements innerHTML textContent setAttribute() removeAttribute() Creating and Inserting Elements createElement() createTextNode() appendChild() insertBefore() Removing Elements removeChild() Event Handling addEventListener() removeEventListener() Common Use Cases Changing Styles Form Validation Dynamic Content Updates Event Delegation ***/
